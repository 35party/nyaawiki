# Don't Starve Together

饥荒（Don't Starve Together）服务器地址在

```
dst.nyaacat.com
```


加入方式:
1. 在游戏主界面按下`~`键打开控制台,输入以下代码:
```
c_connect("dst.nyaacat.com")
```

2. 在联机游戏界面直接搜索 `nyaacat` 也可以看到,不过该方式由于需要klei官方服务器作为代理,会有额外延迟,不推荐,尤其是Steam帐号不在国内的用户。
